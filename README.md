# CAMAD Website

Official website for CAMAD (Community Action Machynlleth and District).

CAMAD supports volunteering, community projects, and local wellbeing initiatives in Machynlleth and the surrounding area. This website provides clear information about services, volunteering opportunities, community support, news, and ways to get involved.

---

## 🌍 About CAMAD

Community Action Machynlleth and District (CAMAD) is a community-focused organisation dedicated to strengthening local connections, supporting vulnerable residents, and encouraging volunteering across the Machynlleth area.

The organisation works to:

- Promote and coordinate volunteering opportunities  
- Support individuals and families in need  
- Facilitate community-led initiatives  
- Strengthen local partnerships  
- Improve wellbeing and social inclusion
